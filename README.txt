Instructions of running this program

1. Compile using -> gcc main.c -o gc
2. Run the executable along with file path as input ->  ./gc dummyFile.txt

Input -> "dummyFile.txt" as command line argument

Output -> should see the text in dummyFile.txt in reverse order


Explanation ->
The dummyFile.txt contains a simple text of "Hello World". On running the program for the first time you should see the text reversed "dlroW olleH". If you execute again it will be again reversed to form the original text of "Hello World".
